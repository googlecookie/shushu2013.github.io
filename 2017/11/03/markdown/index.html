1. 在HTML区块标签间的MarkDown格式语法将不会被处理。
<table>
    <tr>
        <td>*FOO*</td>
    </tr>
</table>
2. 特殊字符自动转换
在HTML文件中，有两个字符需要特殊处理：< 和 & 。
< 符号用于起始标签，& 符号则用于标记HTML实体。
如果要显示这些符号，必须使用实体形式，如 &lt; 和 &amp; 。

在Markdown中，你可以自然地书写字符，需要转换的由它来处理好了。
如果你使用的 & 字符是HTML字符实体的一部分，它会保留原状，否则
它会被转换为 &amp;。
3. 区块元素
3.1 段落和换行
一个Markdown段落是由一个或多个连续的文本组成，它的前后要有一个以上的空行。
段落内换行可以插入换行标签<br/>，也可以在插入处先按入两个以上的空格然后回车。
例如：
这是第一段

这是第二段

这是第一段

这是第二段

这是第一行，注意后面加两个空格，然后回车
这是第二行

这是第一行，注意后面加两个空格，然后回车
这是第二行

这是第一行<br/>这是第二行

这是第一行
这是第二行

3.2 标题
Markdown支持两种标题的语法，类 Settext 和类 atx 形式。
类 Setext 形式是用底线的形式，利用 = （最高阶标题）和 - (第二阶标题)，例如：
    This is an H1
    =============
    This is an H2
    -------------
任何数量的 = 和 - 都可以有效果

This is an H1

This is an H2

类 axt 形式则是在行首插入 1 到 6 个 #，对应到标题 1 到 6 阶，例如：
# 这是 H1
## 这是 H2
###### 这是 H6

这是H1

这是H2

这是H6
3.3 区块引用 Blockquotes
Markdown 标记区块引用是使用类似 email中用 > 的引用方式。
如果你还熟悉在 email 信件中的引言部分，你就知道怎么在Markdown文件中建立一个区块引用，
那会看起来像是你自己先断好行，然后再每行的最前面加上 > 。

>两个黄鹂鸣翠柳，一行白鹭上青天。
>
>窗含西岭千秋雪，门泊东吴万里船。

两个黄鹂鸣翠柳，一行白鹭上青天。

窗含西岭千秋雪，门泊东吴万里船。

区块引用也可以嵌套（例如：引用内的引用），只要根据层次加上不同数量的 > :

>两个黄鹂鸣翠柳，一行白鹭上青天。
>
>窗含西岭千秋雪，门泊东吴万里船。
>>出自唐代诗人杜甫的诗
>>><<绝句>>

两个黄鹂鸣翠柳，一行白鹭上青天。

窗含西岭千秋雪，门泊东吴万里船。

出自唐代诗人杜甫的诗

<<绝句>>

引用的区块也可以使用其他的Markdown语法，包括标题、列表、代码区块等：
>### <<绝句>>
>#### 杜甫
>两个黄鹂鸣翠柳，一行白鹭上青天。

>寒窗西岭千秋雪，门泊东吴万里船。

<<绝句>>

杜甫

两个黄鹂鸣翠柳，一行白鹭上青天。

寒窗西岭千秋雪，门泊东吴万里船。

3.4 列表
Markdown 支持有序列表和无序列表
无序列表使用星号、加号或是减号作为列表标记：
* Red
* Green
* Blue
等同于：
+ Red
+ Green
+ Blue
或者：
- Red
- Green
- Blue
  • Red
  • Green
  • Blue
有序列表则使用数字接着一个英文句点：
1. Bird
2. McHale
3. Parish
很重要的一点是，你在列表标记上使用的数字并不会影响输出的HTML结果，
上面的列表所产生的HTML标记为：
<ol>
<li>Bird</li>
<li>McHale</li>
<li>Parish</li>
</ol>
如果你的列表标记写成：
3. Bird
1. McHale
5. Parish
你都会得到完全相同的HTML输出。这是一种懒惰的写法，建议第一个项目最好还是
从1. 开始，因为Markdown 未来可能会支持有序列表的 start 属性。

1.Bird
2.McHale
3.Parish

如果要在列表项目内放进引用，那 > 就需要缩进：
* A list item with a blockquote:
    >This is a blockquote
    >inside a list item.
  • A list item with a blockquote:

    This is a blockquote
    inside a list item.

如果要放代码区块的话，该区块就需要缩进两次，也就是8个空格或是2个制表符：
* Java 输出 Hello, WWB!

        system.out.print("Hello, WWB!");
  • Java 输出 Hello, WWB!
    system.out.print("Hello, WWB!");
当然，项目列表很可能会不小心产生，像下面这样的写法，即在行首出现数字-句点-空白，
要避免这样的状况，你可以在句点前面加上反斜杠。
1993\. I am birth.

1993. I am birth.

3.5 代码区块
在 Markdown 中建立代码区块很简单，只要简单地缩进 4 个空格或是 1 个制表符就可以，例如：
Java 输出 Hello, WWB!

    system.out.print("Hello, WWB!");

Java 输出 Hello, WWB!

system.out.print("Hello, WWB!");
3.6 分割线
可以在一行中用三个以上的星号、减号、下划线来建立一个分隔线，行内不能有其他东西。
也可以在星号或是减号、下划线中间插入空格。下面每种写法都可以建立分隔线：
* * *
***
*****
- - -
_ _ _ _
4. 区段元素
4.1 链接
Markdown 支持两种形式的链接语法：行内式和参考式两种形式。
不管是哪一种，链接文字都是用 [方括号] 来标记。

行内式的链接：在方括号后[]面紧接着圆括号()，圆括号内是链接地址，如果你还想加上
链接的 title 文字，只要在网址后面，用双引号把 title 文字包起来即可。

这是：[百度链接](https://www.baidu.com) <br>
这是：[百度链接](https://www.baidu.com/ "百度")

这是：百度链接
这是：百度链接

参考式的链接：在链接文字的括号后面再接上另一个方括号，在第二个方括号内填入
用以辨识链接的标记（链接也可以用尖括号<>包起来）。

这是 [百度链接][id]

接着，在文件的任意处，你可以把这个标记的链接内容定义出来，如下所示：
[id]: https://www.baidu.com "百度"

链接内容定义的形式为：
    1.方括号（前面可以选择性地加上至多三个空格来缩进），里面输入链接文字
    2.接着一个冒号
    3.接着一个以上的空格或制表符
    4.接着链接的网址
    5.选择性地接着 title 内容，可以用单引号、双引号或是括弧包着

这是 百度链接

隐式链接标记功能：
[Google][]
然后定义链接内容：
[Google]: https://google.com.hk/ "谷歌"

Google

4.2 强调
Markdown 使用星号（*）和下划线（_）作为标记强调字词的符号，被 * 或 _ 包围的字
词会被转成用 <em> 标签包围，用两个 * 或 _ 包起来的话，则会被转成 <strong>。

*我要强调* <br/>
_我要强调_ <br/>
**我要强调** <br/>
__我要强调__ <br/>

我要强调
我要强调
我要强调
我要强调

强调也可以直接插在文字中间，但是你的 * 和 _ 两边都有空白的话，
它们就只会当做普通的符号，可以用反斜线\来转义，直接显示 * 或 _ ：

我要*强调* <br/>
我要 * 强调 * <br/>
我要\*强调\* <br/>

我要强调
我要 强调
我要*强调*

4.3 代码
如果要标记一小段行内代码，你可以用反引号 ` 把它包起来。例如：
Use the `printf()` function

Use the printf() function

如果要在代码区段内插入反引号，你可以用多个反引号来表示开始和结束代码区段：
`` system.out.print(`) ``

system.out.print(`)

4.4 图片
Markdown 使用一种和链接很相似的语法来标记图片，同样也允许两种样式：
行内式和参考式。
行内式：
    1、一个感叹号！
    2、接着一个方括号[]，里面放上图片的替代文字
    3、接着一个普通括号()，里面放上图片的网址或路径，最后还可以用引号包住
    并加上选择性的 "title" 文字。
例如：
![My IMG](/image/image.jpg)

![My IMG](/image/image.jpg "佐助")


参考式的图片语法则长得像这样，
其中[id]是图片参考的名称，图片参考的定义方式则和链接参考一样：

![My IMG][id]
[id]: /image/image.jpg "佐助"


到目前为止，Markdown 还没有办法指定图片的宽高，如果你需要的话，你可以使用
html的<img>标签。
<img src="/image/image.jpg" height="100px" widht="100px" alt="佐助">

5. 其它
5.1 自动链接
Markdown 支持以比较简短的自动链接形式来处理网址和电子邮箱地址，只要是用尖
括号包起来，Markdown 就会自动把它转为链接。
一般网址的链接文字和链接地址一样，例如：

<https://shushu2013.github.io> <br/>
<125***3436@qq.com>

https://shushu2013.github.io
125***3436@qq.com

5.2 反斜杠
Markdown 可以利用反斜杠来插入一些在语法中有其它意义的符号。
比如你想用星号加在文字旁边,但不想做出强调效果，你可以在星号前加上反斜线：

\*我要星号\*

*我要星号*

Markdown 支持以下这些符号前面加上反斜杠来帮助插入普通的符号：

\ 反斜线
` 反引号
* 星号
_ 下划线
{} 花括号
[] 方括号
() 括号
# 井字号
+ 加好
- 减号
. 英文句号
! 感叹号
